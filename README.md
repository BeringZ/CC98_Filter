### 安装步骤

- 手动安装（开发者模式）（善用搜索说是🙈）
	- 下载releases中的文件并解压
	- 进入Chrome的插件页面
	- 开启右上角的"开发者模式"
	- 点击"加载已解压的扩展程序"
	- 选择解压后的文件夹

### 使用方法

- 打开管理界面​
	- 点击浏览器右上角的扩展图标
	- 弹出用户管理窗口
- ​添加要屏蔽的用户​
	- 在输入框中输入用户名
	- 点击"添加"按钮或按Enter键
	- 用户将立即被加入屏蔽列表
- 管理屏蔽列表​

	- 启用/禁用屏蔽​：使用每个用户右侧的开关
	- ​移除用户​：点击"移除"按钮
 	- 查看屏蔽时间​：显示首次屏蔽和最后屏蔽时间

### 注意
- 屏蔽没有效果可尝试刷新
- 具体屏蔽范围详见开发目录
